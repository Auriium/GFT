# Gunfight Tactics
- An advanced team-based duel core with customizable kits

# Features
- Customizable kits with easy creator
- Plenty of GUIS
- Scoreboard support
- 2 Gamemodes, Bomb and TDM
- SQL support for stats
- Config system
